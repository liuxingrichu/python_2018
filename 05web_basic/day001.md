### 快捷方式 ###
1. 生成html基本模块
	1. "!" + "Tab"（英语状态）
2. 自动补全标签
	1. "a" + "Tab"
	2. "div" + "Tab"
	3. "li*6" + "Tab"
	4. "img" + "Tab"
	5. "tr>td*3" + "Tab"
3. 同时修改多个相同内容
	1. "Ctrl" + "d"
4. 排版
	1. 局部排版
		1. 选中需排版内容
		2. “Shift + Ctrl + p” -> Beautify selection
	2. 整个文件排版
		1. “Shift + Ctrl + p” -> Beautify file
